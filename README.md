# Telegram Бот для "Лиги Азии" (RAG)

Бот предназначен для ответов на вопросы судей, используя регламент Лиги Азии. Он использует OpenAI (GPT-4o-mini) для генерации ответов и Qdrant для поиска по базе знаний.

## 🛠 Технологии
- **Python 3.10+** (Aiogram 3.x)
- **OpenAI API**
- **Qdrant** (Векторная база данных)
- **Redis** (Кэширование ответов и история)
- **Google Docs API** (Загрузка регламента)
- **Docker & Docker Compose**

## 🚀 Установка и запуск

### 1. Предварительные требования
Убедитесь, что у вас установлены:
- [Docker](https://www.docker.com/products/docker-desktop/)
- [Docker Compose](https://docs.docker.com/compose/install/)

### 2. Настройка окружения
1. Скопируйте файл с примерами переменных окружения:
   ```bash
   cp .env.example .env
   ```
2. Откройте файл `.env` и заполните необходимые данные:
   ```ini
   BOT_TOKEN=ващ_токен_бота
   OPENAI_API_KEY=ваш_ключ_openai
   
   # Список ID администраторов (JSON формат)
   ADMIN_IDS=[12345678, 98765432]
   
   # ID Google документа (можно взять из URL документа)
   GOOGLE_DOC_ID=1xYz...
   
   # Если документ закрыт, нужен JSON сервисного аккаунта.
   # Если документ доступен по ссылке (Public), оставьте пустым.
   # GOOGLE_SERVICE_ACCOUNT_JSON={}
   
   # Настройки БД (можно оставить по умолчанию для Docker)
   QDRANT_HOST=qdrant
   REDIS_URL=redis://redis:6379/0
   ```

### 3. Запуск
Запустите проект с помощью Docker Compose:
```bash
docker compose up --build -d
```
Эта команда скачает образы, соберет бота и запустит все контейнеры в фоновом режиме.

Посмотреть логи можно командой:
```bash
docker compose logs -f bot
```

### 4. Использование
1. Найдите бота в Telegram и нажмите **/start**.
2. **Инициализация базы знаний**:
   Чтобы бот скачал текст из Google Doc и сохранил его в базу, отправьте команду (в коде пока нет проверки на админа, доступна всем):
   ```
   /update_kbase
   ```
   Дождитесь сообщения "Обновление завершено".
   *Примечание: Если вы администратор (ваш ID в ADMIN_IDS), используйте команду `/admin` для вызова панели управления.*
3. **Вопросы**:
   Просто отправьте боту вопрос. Например:
   > "Какой штраф за опоздание на игру?"

Бот найдет информацию в регламенте и ответит. Если точного ответа нет, он сообщит об этом.

## 📂 Структура проекта
- `bot/` - Исходный код бота.
  - `handlers/` - Обработчики сообщений.
  - `services/` - Логика работы с OpenAI, Qdrant, GDocs.
- `docker-compose.yml` - Конфигурация контейнеров.
